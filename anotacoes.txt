git -> ferramenta
gitHub -> servidor de repositorios
SCM -> Source Control Management

comandos

Adicionar os arquivos ao INDEX, não adiciona no repositorio, só informa que esta sendo usado.
git add <arquivo> ou git add . (todos)

Enviar uma alteração com todos os arquivos e alterações com um numero e comentário
git commit ou  git commit -m "mensagem"

ver o status, arquivos que não estão adicionados no index, arquivos que já foram adicionados
git status

log de commits
git log

enviar os commits com os arquivos e alterações para o repositorio
git push

inicializa um repositório na maquina
git init

desfazer alterações de um arquivo
git checkout -- <path/arquivo>

desfazer alterações de um arquivo que já foi publicado
git checkout HEAD -- <path/file>


baixa um repositório existente
git clone

baixar as alterações do repositorio
git pull

alterar o repositorio para o estado daquele commit
git checkout <commit> <file>

voltar ao ultimo commit
git checkout master
